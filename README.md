# Поиск человека в видеоархиве
Данная программа ищет человека в видеоархиве по лицу.
Скрипт find_video.py выполняет анализ заданной части архива и сохраняет результаты в файл.
Скрипт analytics.py выполняет анализ файлов в заданной папке. В зависимости от путей этот скрипт нужно отредактировать (там где должны быть пути, прописаны примеры).

## Использование
Для использования данной программы нужно установить python и несколько библиотек. Самая "увесистая" из них - *face_recognition*, которая требует *dlib*, который требует компилятор msbuild для построения.

Настройки find_video.py находятся в 2 файлах: `list.ini` и `set.ini`, которые находятся в той же папке, что и скрипт.

**list.ini** содержит список путей к папкам с видеофайлами, среди которых надо найти видео с заданным человеком (поиск ведётся и в подпапках).

**set.ini** в первой строке содержит путь к фото, где искомый человек ОДИН, повёрнут к камере личом и виден хорошо (это фото - шаблон); во второй строке находится количество потоков поиска (предполагается, что в поиске может быть задействовано несколько несколько скриптов параллельно);
на третьех строке находится индекс текущего потока (для каждого потока должен быть уникальным и не превосходищим по число потоков). Количество потоков и индекс позволяют поделить обработку между потоками наиболее равномерно.

Важно: все потоки, в которых используется разделение по индексу (пусть будут называться группами поиска) должны иметь одинаковые списки путей к папкам с видеофайлами. Группы НЕ должны пересекаться.
